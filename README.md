# Mobile Prompt Bridge MVP

スマホをPCのテキスト入力端末として使い、同一Wi-Fi環境からPCで起動している **Google Antigravity IDE**（またはその他のエディタ）のチャット欄に、プロンプトを瞬時に貼り付け・送信するための補助ツールです。

リモートデスクトップを使わず、スマホのフリック入力や音声入力をそのままPCでのプロンプト入力に利用することができます。

---

## 🛠️ 前提条件
- **OS**: Windows 10 / 11
- **Python**: 3.11 以上がインストールされていること
- **ネットワーク**: PCとスマホが同じWi-Fi（ルーター）に接続されていること
- **対象エディタ**: Google Antigravity IDE や VS Code などがPC側で起動していること

---

## 📦 インストール方法

### 手動でインストールする場合
1. このリポジトリを `git clone` するか、ZIPでダウンロードして解凍します。
2. コマンドプロンプトまたはPowerShellを開き、ダウンロードしたプロジェクトのフォルダ（`Mobile-Prompt-Bridge` の中）に移動し、必要なライブラリをインストールします。
   ```bash
   cd 任意のパス/Mobile-Prompt-Bridge
   pip install -r requirements.txt
   ```

### 🤖 AIエージェントに全自動で導入させる場合（おすすめ！）
Cursor、Windsurf、Antigravity IDE などのエージェント型AIをお使いの場合は、面倒なセットアップを全てAIに任せることができます。
1. エディタで `agent_setup_prompt.md` ファイルを開きます。
2. 中に書かれているプロンプトをコピーし、ご自身のAIアシスタントにチャットで送信してください。
3. Pythonパッケージのインストールから起動、接続URLの発行までをAIが全自動で行ってくれます。

## ⚙️ 設定ファイル (`config.json`) の解説

本アプリを起動すると、フォルダ内に自動的に `config.json` が生成されます。必要に応じてテキストエディタで開いてカスタマイズできます。

```json
{
  "port": 8712,
  "allowed_window_titles": [
    "antigravity",
    "visual studio code",
    "cursor",
    "windsurf"
  ],
  "send_key": "Enter",
  "enable_auto_send": false
}
```

- **`port`**: Webサーバーを起動するポート番号（デフォルト: `8712`）。
- **`allowed_window_titles`**: 貼り付けを許可するウィンドウのタイトルに含まれる文字列のリスト（大文字小文字は区別されません）。
  - *セキュリティと誤爆防止のため、このリストに含まれないウィンドウ（ブラウザ、ブラウザでのログイン画面、ファイルエクスプローラー等）が最前面にある場合は、スマホからの貼り付け要求が安全に拒否されます。*
- **`send_key`**: 貼り付けた後の送信操作に使うキー（`"Enter"` または `"Ctrl+Enter"`）。
- **`enable_auto_send`**: スマホ側からの「貼り付けて送信」ボタンを有効にするかどうかのフラグ（デフォルト: `false`）。
  - *誤送信を防ぐため、初期設定では無効化されています。自動送信まで行いたい場合は、ここを `true` に変更してください。*

## 🚀 使い方（ngrokを使った外部公開）

外出先や別ネットワークのスマホからアクセスするには、安全なトンネリングツール「**ngrok**」を使用します。

### ステップ1: サーバーの起動
1. プロジェクトフォルダにある **`run.bat`** をダブルクリックして起動します。
2. コマンドプロンプトが開き、サーバーが `http://127.0.0.1:8712` で起動します。
   ※この時点では画面に表示されるローカル用URL（`http://192.168...`）は無視して構いません。セキュリティトークン（`?token=...`の部分）だけ後で使うのでメモしてください。

### ステップ2: ngrokでトンネルを開通する
1. 別のコマンドプロンプト（またはPowerShell）を開きます。
2. 以下のコマンドを実行し、ポート `8712` をインターネット上に安全に公開します。
   ```bash
   ngrok http 8712
   ```
3. 実行すると画面上に `Forwarding  https://xxxxx.ngrok-free.app -> http://localhost:8712` のようなURLが表示されます。

### ステップ3: スマホからアクセスする
1. スマホのブラウザで、ngrokが発行したURLに先ほどのトークンをつけてアクセスします。
   👉 `https://xxxxx.ngrok-free.app/?token=あなたのトークン`
2. PCの画面上で **対象エディタ (Antigravity IDE など)** のウィンドウを開いておきます。
3. スマホの画面でテキストを入力し、**「Antigravityへ貼り付ける」** などのボタンをタップします。
4. PC上のウィンドウが自動的に前面化され、操作が実行されます！

## 🔒 セキュリティについて
- LAN内（プライベートネットワーク）のみで動作する前提で設計されています。ポート開放やngrokなどの外部公開は不要で、インターネット上からのアクセスは完全に遮断されます。
- 起動するたびに**ランダムなセキュリティトークン**が自動生成され、URLに付与されます。このトークン（`token`）が一致しないアクセスはすべて拒否されます。
- 許可リストにないウィンドウへの誤貼り付けを防ぐセキュリティガード機能が標準で機能しています。

## 🔗 URLとトークンを完全に固定する手順（推奨）

通常、起動するたびに「トークン」と「ngrokのURL」がランダムに変わってしまいます。
以下の設定を行うことで、**完全に同じ固定URL**をスマホに登録し、永続的に使い続けることができます。

### ① セキュリティトークンの固定
1. プロジェクトフォルダの `config.json` を開きます。
2. 最下部にある `"security_token": ""` を、**お好みのパスワード（半角英数字、例: "mysecrettoken"）** に書き換えて保存します。
   ```json
   "security_token": "mysecrettoken"
   ```
3. これにより、サーバーを再起動してもトークン値は常に設定した値に固定されます。

### ② ngrok 公開ドメインの固定化（無料）
1. **[ngrokダッシュボード](https://dashboard.ngrok.com/)** にログインします。
2. 左メニューから `Cloud Edge` -> `Domains` を開き、**`Create Domain`** をクリックして無料の固定ドメイン（例: `yourname.ngrok-free.app`）を1つ生成・確保します。
3. 今後、PC側で ngrok を起動する際に、以下のように `--domain` オプションを付けて実行します：
   ```bash
   ngrok http 8712 --domain=あなたの固定ドメイン名
   ```

### 📱 最終的な固定URL
上記①・②を設定すると、スマホからアクセスするURLは永久に以下に固定されます：
👉 `https://あなたの固定ドメイン名/?token=あなたの固定トークン`
*(例: `https://yourname.ngrok-free.app/?token=mysecrettoken`)*

このURLをスマホのブラウザでホーム画面に追加（ブックマーク）しておけば、今後はPC側でバッチとngrokを動かすだけで、一切の入力変更なしで常に一発接続できるようになります。

## 🌟 新機能について

### ③ チャット開閉状態の自動判定（キャリブレーション）
スマホのWeb画面に「チャット開閉判定の設定」機能が追加されました。
PC側でチャットを「開いた状態」と「閉じた状態」にして登録（キャリブレーション）することで、PC上のピクセルカラーの変化を学習し、自動で状態を判別します。
- **チャットが閉じている時**: トグルキーを1回送信して開き、入力します。
- **チャットが開いている時**: トグルキーを2回（閉じ ➡️ 開き直し）送信し、エディタ等にカーソルが逃げていても確実にチャット欄にフォーカスを合わせてから入力します。

### ④ AI対話ログの同期表示
スマホの画面上で、これまでの会話履歴（自分とAIのやり取り）がチャットの吹き出し形式で表示されます。
送信後にAIが回答している間は、スマホ側にもローディングアニメーションが表示され、回答が完了すると自動的に履歴が同期されます。これにより、PCの画面を見ずにスマホだけでAIとのやり取りを完結できます。
