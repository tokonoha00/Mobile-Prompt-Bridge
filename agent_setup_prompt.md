# Mobile Prompt Bridge 導入・セットアップ依頼プロンプト

このファイルは、Cursor、Windsurf、Antigravity IDE などの「ローカル環境で動作するエージェント型AI」に読み込ませて、環境構築を全自動でやってもらうためのプロンプトです。

以下のテキストをコピーして、あなたのAIアシスタントに送信してください。

## ✂️ コピー用プロンプト

あなたは私のローカル環境で動作する優秀なAIエージェントです。
現在開いているワークスペース（`Mobile-Prompt-Bridge`）のセットアップと起動を自動的に行ってください。

### 実行してほしいタスク
1. **Python環境の確認**: Python 3.11以上がインストールされているか確認してください。
2. **依存関係のインストール**: ターミナルで `pip install -r requirements.txt` を実行し、必要なパッケージをインストールしてください。
3. **一括起動スクリプトの実行**: 準備が完了したら、`start_all.bat` を**必ずユーザーが見える統合ターミナル（Interactive Terminal）内で実行**してください（バックグラウンドの非表示タスクとして実行すると、UI Automationが正常に動作しない Session 0 問題が発生するためです）。実行によりローカルサーバーとngrokが別ウィンドウで立ち上がります。
4. **接続URLの特定と報告**: ngrokのターミナル出力から公開URL（`https://*.ngrok-free.app`）を取得し、サーバー側で生成された `config.json` に記載されている `security_token` を組み合わせて、私がスマホからアクセスするための完全なURL（例: `https://xxx.ngrok-free.app/?token=yyy`）を作成して教えてください。
5. **AutoAccept拡張の案内**: スマホから遠隔操作する際、IDEの実行確認ポップアップでブロックされないよう、VSCode拡張機能の `AntiGravity-AutoAccept` をインストールすることが推奨されています。そのインストール方法について私に案内してください。

エラーが出た場合は、自律的に原因を調査し、修正を試みてください。
起動と確認が完了したら、最終的なアクセスURLと「スマホでの操作準備が整ったこと」を私に報告してください。
